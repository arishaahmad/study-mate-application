# study-mate-application
An application designed to help students manage their studies, schedules, and tasks efficiently.

Core Features Implemented
Authentication: Secure registration and login using Supabase Auth.
Class Schedule: Users can add, view, and delete course details.
Pomodoro Timer: Logic for users to start, pause, and reset study sessions.
To-Do List: Functionality to add new tasks with due dates.
